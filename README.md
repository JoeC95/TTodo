# TTodo

## 介绍

一个记录任务，规划时间的微信小程序。

## 功能

- 添加任务，设置任务截止时间、分类、紧急状况、是否为日常、任务名、任务描述
- 标记为已完成
- 删除任务
- 统计已完成任务数
- 展示未完成任务

## 部署

1. clone 到本地
> git clone git@github.com:JoeC95/TTodo.git
2. 从微信小程序开发ide 中打开此文件夹即可
